# 오성정공 금형기반 스마트 생산품질 관리시스템

## 1. 프로젝트 개요
본 시스템은 자동차 프레스 부품 제조현장에서 금형 상태 데이터를 기반으로 생산, 품질, 정비 정보를 통합 관리하기 위한 금형기반 스마트 생산품질 관리시스템입니다.  
기존 제조현장에서 생산관리, 품질관리, 금형관리, 정비이력이 개별적으로 관리되거나 수기·경험 중심으로 운영되는 문제를 개선하고, 데이터 기반 품질혁신 및 예방정비 체계를 구축하는 것을 목표로 개발되었습니다.

---

## 2. 개발 배경
자동차 부품 제조업에서는 금형 상태가 제품 품질에 직접적인 영향을 미칩니다.  
그러나 중소 제조현장에서는 다음과 같은 문제가 빈번하게 발생합니다.

- 금형 수명 및 누적 타수 관리 미흡
- 생산 LOT별 품질 이력 추적 한계
- 불량 발생 원인과 금형 상태 간 연계 분석 부족
- 예방정비보다 사후 대응 중심의 운영
- 생산, 품질, 정비 데이터의 분절 관리

본 시스템은 이러한 문제를 해결하기 위하여 금형, 생산, 불량, 검사, 정비 데이터를 통합 연결하는 구조로 설계되었습니다.

---

## 3. 시스템 구성
본 시스템은 다음의 모듈로 구성됩니다.

- 품목 관리
- 금형 관리
- 생산계획 관리
- 작업지시 관리
- 생산실적 및 LOT 관리
- 불량 등록 관리
- 순회검사 관리
- 품질 트렌드 분석
- 정비이력 관리

---

## 4. 주요 기능

### 4.1 품목 관리
- 품번, 품명, 납품처, 표준 사이클 등록
- 기준치수(Nominal), 상한(USL), 하한(LSL) 등록
- 품질검사 기준 데이터의 기초정보 관리

### 4.2 금형 관리
- 금형코드, 금형명, 적용 품번, 수명타수, PM주기 등록
- 현재 누적 타수 및 최종 PM 완료 타수 관리
- 금형 상태 및 보관 위치 관리

### 4.3 생산계획 및 작업지시
- 품목 및 금형 기준 생산계획 수립
- 생산계획 기반 작업지시 생성
- 지시 수량 및 예정일시 관리

### 4.4 생산실적 및 LOT 관리
- 작업지시 기준 실적 입력
- 실적 입력 시 LOT 자동 생성
- 생산수량, 불량수량, 작업자, 금형 타수 자동 증가 관리

### 4.5 불량 등록
- LOT, 품목, 금형과 연결된 불량이력 등록
- 불량유형, 원인, 조치사항 기록
- 금형별 품질문제 추적 가능

### 4.6 순회검사
- 검사일, 검사시간, 품목, 금형, LOT 기준 검사 등록
- 기준값, 허용상한, 허용하한, 실측값 입력
- PASS / FAIL 자동 판정

### 4.7 품질 트렌드 분석
- 품목별 불량 추이 분석
- 금형별 불량률 분석
- LOT 기반 품질이력 확인
- 금형 수명율과 불량률 간 연계 분석

### 4.8 정비이력 관리
- 금형별 정비일자, 정비유형, 작업자, 비용, 조치내용 기록
- 예방정비 이력 관리
- 품질문제 발생 이후 정비조치 이력 확인 가능

---

## 5. 기술 차별성
기존 생산관리 시스템은 생산 중심 또는 품질 중심으로 기능이 분리되는 경우가 많습니다.  
본 시스템은 다음과 같은 차별성을 가집니다.

- 금형을 중심으로 생산, 품질, 정비 데이터를 연결
- LOT 단위 생산 및 불량 추적 가능
- 검사 기준과 실측값에 따른 자동 판정 기능 제공
- 금형 타수 증가와 정비이력 관리 연동
- 금형 상태와 품질문제의 상관관계 분석 가능

즉, 본 시스템은 단순 생산관리 시스템이 아니라 **금형 기반 품질혁신 및 예방정비 의사결정 지원 시스템**이라는 점에서 차별성을 가집니다.

---

- Flask 기반 웹 시스템
- 로컬 환경에서 `app.py` 실행

5. 생산실적 입력 및 LOT 자동 생성
6. 불량 등록
7. 순회검사 등록
8. 품질 트렌드 분석 확인
9. 정비이력 등록 및 정비이력 관리 확인

금형 상태 데이터를 활용한 생산·품질·정비 통합관리 구조를 구현한 제조업 특화형 시스템입니다.
