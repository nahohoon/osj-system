오성정공 금형기반 스마트 생산품질 관리시스템 (MB-SPQMS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 기술명
  오성정공 금형기반 스마트 생산품질 관리 소프트웨어 기술

□ 기술 분류
  정보통신·소프트웨어 / 제조 스마트화 / 품질관리 정보시스템

────────────────────────────────────────────────────────────
1. 기술 개요
────────────────────────────────────────────────────────────

  자동차 프레스 부품 제조 현장(오성정공)의 현황:
  - 현대자동차 지급 금형을 이용한 고정밀 브라켓 부품 제조
  - 일 생산량 8,000~10,000개 / 대표 직접 순회검사 체계
  - 기존 문제: 금형 타수-수명 관리 수기, 품질 데이터 산발적 기록

  본 기술은 상기 현장 문제를 해결하기 위해 개발한 소프트웨어로서,
  금형 수명 데이터를 핵심 축으로 생산·품질 정보를 통합 연계하는
  자동차 프레스 부품 특화 스마트 생산품질 관리 플랫폼임.

────────────────────────────────────────────────────────────
2. 기술의 차별성 및 혁신성
────────────────────────────────────────────────────────────

  ① 금형 타수 자동 누적 알고리즘
     - 생산실적 입력 시 금형-생산계획-작업지시 연결체계를 통해
       금형 누적타수가 자동 계산·갱신됨
     - 기존: 수기 기록으로 인한 오차 및 관리 누락 빈번
     - 개선: 실시간 정확한 수명 사용률(%) 자동 산출

  ② 예방정비(PM) 자동 알림 엔진
     - PM주기·최종PM타수·현재타수를 기반으로 잔여타수 실시간 계산
     - 잔여 5,000 이하 시 전체 화면에 즉시 경고 표시
     - 기존: 경험적 판단에 의존하여 예방정비 시점 불확실
     - 개선: 데이터 기반 정비 시점 자동 감지로 긴급 수리 예방

  ③ 자동 PASS/FAIL 판정 엔진 (순회검사)
     - 품목별 USL/LSL 기준치 등록 → 검사 시 자동 불러오기
     - 측정값 입력 즉시 항목별 PASS/FAIL 판정 및 종합 결과 집계
     - 기존: 수기 측정·수기 기록으로 오판정·이력 유실
     - 개선: 객관적 자동 판정, 검사 데이터 완전 디지털화

  ④ 금형 수명율 × 불량률 상관 분석
     - 금형별 누적 생산·불량 데이터 기반으로 불량률 자동 계산
     - X축(금형 수명 사용률) × Y축(불량률) 산점도 자동 생성
     - 금형 노후화와 품질 저하의 상관관계를 시각적으로 입증
     - 이를 통한 선제적 금형 교체·정비 의사결정 지원

  ⑤ LOT 자동 채번 및 3차원 품질이력 추적
     - 생산실적 저장 시 타임스탬프 기반 LOT 번호 자동 채번
     - 불량·검사 기록에 LOT번호·금형ID·품목ID 동시 저장
     - LOT 단위, 금형 단위, 품목 단위 품질이력 교차 추적 가능

────────────────────────────────────────────────────────────
3. 기술의 독창성
────────────────────────────────────────────────────────────

  본 시스템은 범용 MES(제조실행시스템) 또는 ERP와 구별되는 아래 특성을 가짐.

  구분            범용 MES/ERP         본 시스템 (MB-SPQMS)
  ──────────────  ─────────────────    ────────────────────────────────
  핵심 구조       생산 중심            금형 수명 데이터가 핵심 축
  금형 관리       선택적 모듈          생산·품질 연계의 필수 기반
  타수 관리       수동 또는 연동       생산실적 저장 시 자동 누적
  PM 알림         별도 설정            잔여타수 실시간 자동 감지
  품질 판정       외부 연동 필요       USL/LSL 기반 자동 PASS/FAIL
  분석 관점       생산량·납기          금형상태 × 불량률 상관관계

  → 오성정공 현장 프로세스에 특화된 소프트웨어로서
    범용 제품으로 대체 불가한 독자적 기술자산임

────────────────────────────────────────────────────────────
4. 기술의 활용 및 기대 효과
────────────────────────────────────────────────────────────

  ① 품질 클레임 대응력 향상
     → LOT번호·금형ID 기반 소급 추적으로 현대자동차 클레임 시
       원인 금형·생산 시점을 5분 이내 특정 가능

  ② 금형 정비 비용 절감 (예측 예방정비)
     → PM 자동 알림으로 긴급 수리 방지, 계획 정비 전환
       연간 긴급 수리 횟수 감소 및 금형 수명 연장 기대

  ③ 품질 데이터 체계화 → 향후 AI 분석 기반 마련
     → 누적 데이터를 기반으로 금형 수명-품질 예측 모델 적용 가능성

  ④ 벤처기업·이노비즈 인증 기술자산
     → 자체 개발 소프트웨어로 기술혁신 실적 증빙

────────────────────────────────────────────────────────────
5. 소프트웨어 구성 요약
────────────────────────────────────────────────────────────

  개발언어    Python 3.x (Flask 프레임워크)
  DB          SQLite (9개 테이블, 관계형 구조)
  화면(UI)    HTML5 / CSS3 / JavaScript (Chart.js 4.x)
  주요 모듈   생산관리(4), 금형관리(2), 품질혁신(2), 기준정보(1)
  총 라우팅   30+ 엔드포인트 (GET/POST)
  코드 규모   Python 900+ 라인 / HTML 2,000+ 라인 / CSS 350+ 라인
